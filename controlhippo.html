CASE STUDY · 03

CONTROLHIPPO

SAAS · Website + Web Application

End-to-end design and development for an omnichannel conversational AI platform — WordPress website for marketing and lead generation, plus a React web application for managing unified messaging, AI agents, and customer analytics.

Role

Full Stack Designer

Industry

SaaS / AI

Stack

WordPress · React · Figma

Overview

THE CHALLENGE

ControlHippo needed a high-converting marketing website to position their omnichannel AI platform in a competitive SaaS market. The site had to clearly communicate complex AI features — unified inbox, conversational agents, automated workflows — while driving demo bookings and sign-ups.

THE SOLUTION

I designed and developed a dynamic WordPress website with conversion-optimized landing pages, clear feature breakdowns, and strategic CTAs. Built with a modular approach to support rapid iteration on messaging, pricing pages, and industry-specific use cases.

Design Process

Competitive & UX Research

Analyzed competing omnichannel platforms to identify positioning gaps and studied user journeys to optimize the visitor-to-lead conversion funnel.

Wireframes & Information Architecture

Structured the sitemap around key product pillars — AI agents, omnichannel inbox, automated workflows, and analytics — ensuring each feature had clear, dedicated pages.

High-Fidelity UI Design

Created polished Figma designs with interactive product previews, animated feature sections, and trust-building elements like testimonials and integration logos.

WordPress Development

Built the front-end in WordPress with custom templates, optimized for SEO, page speed, and easy content management by the marketing team.

CRO & Analytics Setup

Implemented conversion tracking, heatmaps, and A/B test-ready landing pages to continuously improve demo booking rates.

Optimized Core Web Vitals, structured data, and semantic HTML for organic growth.

Overview

THE CHALLENGE

The ControlHippo web application needed to unify conversations from WhatsApp, Instagram, Telegram, Facebook Messenger, and more into a single dashboard. The UI had to handle real-time messaging, AI agent configuration, workflow automation, and analytics — all without overwhelming the user.

THE SOLUTION

I designed the complete UI/UX in Figma and built the front-end in React, creating an intuitive dashboard with a unified inbox, drag-and-drop workflow builder, AI agent management panels, and real-time analytics dashboards that scale across omnichannel operations.

Design & Dev Process

User Flow Mapping

Mapped critical workflows — omnichannel inbox navigation, AI agent setup, automated workflow creation, and analytics review — to minimize friction.

Design System & Components

Built a comprehensive design system in Figma with reusable components for chat interfaces, channel selectors, workflow nodes, and data visualizations.

Dashboard UI Design

Designed the unified inbox with channel filtering, conversation threading, AI-suggested replies, and customer context panels for agent efficiency.

React Front-End Build

Developed the application front-end in React with real-time WebSocket messaging, state management, and responsive layouts for desktop and tablet.

Iteration & QA

Conducted usability testing with sales and support teams, iterating on conversation UIs and workflow builders based on real usage patterns.

Key Deliverables

01

Unified Inbox UI

Omnichannel conversation view with channel filtering, AI replies, and customer context.

02

AI Agent Builder

Configuration panels for sales and support AI agents with training and behavior settings.

03

Workflow Builder

Visual drag-and-drop workflow automation for lead routing, follow-ups, and notifications.

04

Analytics Dashboard

Real-time dashboards with CSAT scores, response times, lead metrics, and AI performance.
