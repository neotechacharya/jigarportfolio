UI/UX design and front-end development for a leading B2B software discovery platform — designing and building category listing pages and product profile pages that help businesses find and compare software solutions.

Role

UI/UX & Front-End Dev

Industry

B2B / SaaS

Stack

Yii PHP · Bootstrap · Figma

Overview

THE CHALLENGE

SoftwareSuggest's category pages needed to surface hundreds of software products in an intuitive, scannable format. Users struggled to compare solutions quickly, leading to high bounce rates and low engagement with the listing content.

THE SOLUTION

I redesigned the category listing pages with a structured layout featuring clear filtering, comparison-friendly cards, and prominent review data. Built the front-end with Bootstrap on Yii PHP, ensuring fast load times across hundreds of dynamically generated category pages.

Design Process

User Behavior Analysis

Studied how users browse and filter software categories, identifying key decision-making patterns — pricing, ratings, feature sets, and peer reviews.

Information Architecture

Structured category page layouts with smart filtering (by features, pricing, deployment), sortable lists, and quick-compare functionality to reduce decision fatigue.

UI Design in Figma

Created high-fidelity designs for listing cards with rating badges, pricing tiers, and feature highlights — ensuring visual consistency across 500+ category pages.

Front-End Development

Built responsive listing pages using Bootstrap on Yii PHP framework with dynamic filtering, lazy loading, and SEO-optimized markup for organic traffic.

Performance & SEO Optimization

Optimized page speed for data-heavy listings, implemented structured data for search engines, and ensured mobile-first responsive behavior.

Key Deliverables

01

Listing Page UI

Redesigned category listing layouts with comparison-friendly product cards and filter systems.

02

Smart Filters

Multi-criteria filtering by pricing, features, deployment type, and user ratings.

03

Responsive Build

Bootstrap-based responsive front-end integrated with Yii PHP backend templates.

04

SEO Markup

Structured data and semantic HTML for improved search engine visibility across categories.

Page Performance

With hundreds of data-heavy listing pages, performance optimization was critical. Every page was tuned for Core Web Vitals — optimizing Largest Contentful Paint (LCP), minimizing Cumulative Layout Shift (CLS), and ensuring responsive First Input Delay (FID) for a fast, seamless browsing experience across all devices.

Overview

THE CHALLENGE

Product profile pages are where users make their final buying decisions. The existing pages were information-dense but poorly structured, making it hard for users to evaluate software features, read reviews, and compare pricing at a glance.

THE SOLUTION

I redesigned the product profile pages with a clear visual hierarchy — hero section with key metrics, tabbed content for features/pricing/reviews, and prominent CTAs. Built the responsive front-end using Bootstrap, ensuring consistency across thousands of dynamically generated product pages.

Design & Dev Process

Content Audit & User Research

Mapped all data points displayed on product pages — features, pricing plans, screenshots, reviews, alternatives — and prioritized them by user importance.

Visual Hierarchy Design

Designed a scannable layout with a product hero section, rating summary, and tabbed navigation for features, pricing, reviews, and alternatives.

UI Component Design

Created reusable components in Figma — review cards, pricing comparison tables, feature checklists, screenshot galleries, and CTA modules.

Bootstrap Front-End Build

Developed the responsive front-end using Bootstrap grid and components, integrated with Yii PHP templates for dynamic content rendering.

Cross-Browser Testing

Ensured pixel-perfect rendering across browsers and devices, with performance optimization for pages with heavy review and media content.

Key Deliverables

01

Profile Page UI

Complete product profile page redesign with structured content sections and clear CTAs.

02

Review System UI

Redesigned review display with rating breakdowns, verified badges, and helpful vote system.

03

Pricing Tables

Comparison-ready pricing tables with feature breakdowns and plan highlights.

04

Responsive Build

Mobile-first Bootstrap implementation across thousands of dynamically rendered product pages.
