CALLHIPPO

SAAS · Website + Web Application

A comprehensive design and development engagement spanning WordPress website creation, UI/UX design for conversion optimization, and React-based web application front-end development.

Role

Full Stack Designer

Industry

SaaS / VoIP

Stack

WordPress · React · Figma

Overview

THE CHALLENGE

CallHippo needed a high-converting marketing website that clearly communicated their VoIP product offerings, built on WordPress for easy content management. The existing site had low engagement and poor conversion metrics that needed to be addressed through strategic design and development.

THE SOLUTION

I designed and developed a complete WordPress website with a focus on conversion-optimized UI/UX. Every page was crafted to guide users through the sales funnel with clear CTAs, engaging visuals, and persuasive layout patterns that improved both engagement and conversion rates.

Design Process

Research & Competitive Analysis

Analyzed competitor websites in the VoIP/SaaS space, identified user pain points, and mapped out the ideal customer journey from landing page to conversion.

Wireframing & Information Architecture

Created detailed wireframes in Figma to establish the page hierarchy, content flow, and CTA placement strategy optimized for maximum user engagement.

Visual Design & Prototyping

Developed high-fidelity UI designs with a modern, professional aesthetic that aligned with CallHippo's brand identity. Created interactive prototypes for stakeholder review.

WordPress Development

Built the complete website in WordPress with custom themes and responsive layouts. Ensured fast loading speeds, SEO optimization, and easy content management for the team.

Testing & Conversion Optimization

Conducted A/B testing on key landing pages, refined CTAs, and optimized the user flow to improve conversion rates and reduce bounce rates.

Page Performance

Performance was a key focus throughout the build. Every page was optimized for Core Web Vitals — minimizing Largest Contentful Paint (LCP), reducing Cumulative Layout Shift (CLS), and ensuring fast First Input Delay (FID) to deliver a smooth, high-performing experience across all devices.

Overview

THE CHALLENGE

The CallHippo web application needed a modern, intuitive interface that could handle complex VoIP workflows while remaining accessible to users of all technical levels. The existing application UI was cluttered and led to high support ticket volumes.

THE SOLUTION

I redesigned the entire web application UI/UX in Figma and built the front-end in React. The new interface featured a streamlined dashboard, simplified call management flows, and a component-based architecture that ensured consistency and scalability.

Design & Dev Process

User Research & Audit

Analyzed existing user flows, identified friction points in the call management workflows, and gathered feedback from both support teams and end users.

Design System Creation

Built a comprehensive design system in Figma with reusable components, tokens, and guidelines to ensure visual consistency across the entire application.

UI/UX Design & Prototyping

Designed all application screens — dashboard, call logs, contacts, settings, and analytics — with interactive Figma prototypes for usability testing.

React Front-End Development

Developed the application front-end in React with modular, reusable components. Implemented state management, API integrations, and real-time data updates.

QA & Iteration

Conducted thorough cross-browser testing, performance optimization, and iterative design improvements based on user feedback and analytics data.
